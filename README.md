<h2>Clique na tela:</h2>
- o desafio pedia para criar uma interface onde ao clicar aparecesse um circulo </br>
- deveria ter dois botoes: um para desfazer a acao e outro para refazer a acao </br>
<h3>- Logica utilizada</h3> -> usei a estrutura de dados em pilha para armazenar todos os itens adicionados na pilha de desfazer, e ao clicar em refazer, usando o principio LIFO, o ultimo item da pilha desfazer iria para a pilha refazer, podendo haver essa troca de items sem perder o historio de mudancas
